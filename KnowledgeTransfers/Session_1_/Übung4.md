## Übung 4: Policy-Zuweisung und Device Status prüfen

### Ziel

Die Teilnehmer sollen nachvollziehen können, ob eine Intune-Policy auf einem Gerät angekommen ist.

### Schritte

1. Öffne die relevante Endpoint-Security-Policy in Intune.
2. Prüfe:
   - Assignments
   - Included groups
   - Excluded groups
   - Device status
   - User status
   - Per-setting status, falls verfügbar

### Prüfpunkte

| Prüffrage | Ergebnis |
|---|---|
| Ist die richtige Gerätegruppe zugewiesen? |  |
| Gibt es ausgeschlossene Gruppen? |  |
| Gibt es Geräte mit Fehlerstatus? | |
| Gibt es Konflikte mit anderen Policies? |  |
| Ist der Status aktuell? |  |

### Merksatz

Ein erfolgreiches Assignment bedeutet nicht automatisch, dass alle Einstellungen lokal bereits wirksam sind. Intune-Reporting und lokale Clientprüfung sollten gemeinsam betrachtet werden.
