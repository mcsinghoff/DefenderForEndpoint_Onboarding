## Übung 3: Ein Testgerät in Intune suchen

### Ziel

Wir wollen nun ein Gerät in Intune finden und dessen Verwaltungszustand prüfen können.

### Schritte

1. Öffne das Intune Admin Center.
2. Navigiere zu:

   `Devices` → `Windows` → `Windows devices`

3. Suche unser Pilotgerät. (W11PC9610)
4. Öffne das Gerät.
5. Prüfe:
   - Gerätename
   - Benutzer
   - Compliance-Status
   - Management Name
   - Ownership
   - Join Type
   - Last check-in
   - Primary user
   - Device configuration
   - Endpoint security configuration

### Diskussionsfragen

- Ist das Gerät aktuell erreichbar?
- Wann war der letzte Intune Check-in?
- Ist das Gerät compliant?
- Welche Konfigurationsprofile sind angewendet?
- Welche Endpoint-Security-Policies sind angewendet?
